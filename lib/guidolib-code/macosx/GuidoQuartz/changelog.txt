Version 0.2.0 - 05-11-2004
==============================================

- GuidoLib engine version 1.2.2:
	¥ Better automatic positioning for ties and slurs, articulations
	  (staccato, accent...), and tuplets. 
	¥ Octava clefs and double treble clefs.
	¥ Pre-implementation of the \accol tag. Guido can now display the 
	  "straightBrace" accolade.
	¥ Text positionning ( \text tag ) implementation closer to the 
	  Guido specifications. 
	¥ Introduction of Longa notes, diamond and cross heads, and
	  breath-marks.
	¥ Fixed the intens "ff" and "pp" bugs.
	¥ Improved overall speed.

- Alert boxes for parse or file errors.
- Faster rendering.
