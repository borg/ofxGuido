5.1 \glissando

\glissando<style>

Marks glissandos, affects both playback and graphic appearance. >>>add dx1,dy1,dx2,dy2 (thickness) (text via \text tag)

Parameters (in the order of their default positions):

  • style - graphic apperance of glissando indication

    • standard values:
    • "line" - mark glissando with straight line
    • "wavyLine" - marks glissando with wavy line
    default value: "line"
Range: obligatory

Semantics:

  • Indicates a glissando between the outermost notes of its range; if the range contains more than two notes, the inner notes are interpreted as transitory notes in a glissando chain.

 \orn<style,speed,dx,dy,size>(chords)

  • style - graphic apperance of ornament

    • standard values:
    • "turn"
    • "invTurn"
    • "mord"
    • "invMord"
    default value: "turn"
  • speed -
  • dx - horizontal distance between center of symbol and center of note head, in halfspaces of the current system.

    • default value: 2.0
  • dy - vertical distance between center of symbol and center of note head, in halfspaces of the current system.

    • default value: none, see Semantics.
  • size  - of the symbol, as a factor relative to the size of the note head.

    • default value: 1.0

5.11 \figured

This tag marks figures bass notation.

  \figured<fig>(notes)
  \figured<fig,dy>(notes)

Parameters (in the order of their default positions):

  • fig -  a string containing a number of the following

    •   symbols, seperated by commata:
    • n (1,2,...)
    • n/ (slashed n)
    • &,#,n (flat,sharp,natural) and combinations of those
    • "-" to indicate dash for corr figure on that note / range
    • n+
    e.g.: "3,5" "-,5/,6&" "4,-,n"
      -> 3 is extended over first two chords,
         5 over last two chords
Range: notes

Semantics:
