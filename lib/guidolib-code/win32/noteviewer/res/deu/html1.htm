Introcution

gmnview32 is a tool that displays and prints GUIDO Music Notation (gmn) files on Windows 95 and Windows NT 4.0 (or higher versions).

GUIDO Music Notation is a textbased, humanreadable music representation format, that not only allows to specify notational aspects but also musical information for a given piece. GUIDO Music Notation is being developed by the SALIERI Project Group at the Darmstadt University of Technology.

For more information on GUIDO and the SALIERI Project Group please visit our homepage at http://www.informatik.tu-darmstadt.de/AFS/GUIDO

http://sourceforge.net/projects/guidolib/

Getting help

Version 0.91 of gmnview32 is a prototype. gmnview32 is constantly being enhanced, so please visit our website regularly to keep informed of updates.

You can also send email to get help on GUIDO and gmnview32.
